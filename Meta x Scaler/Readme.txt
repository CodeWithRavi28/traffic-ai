Project: Emergency-Aware Adaptive Traffic Signal using Reinforcement Learning

Description:
This project simulates an intelligent traffic control system using reinforcement learning. 
The system dynamically selects traffic signals based on traffic density and prioritizes 
emergency vehicles using reward-based learning (Q-values). The model improves over time 
and its learning behavior is visualized using graphs.

How to run:
1. Install Python
2. Run: python main.py
